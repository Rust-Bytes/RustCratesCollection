# RustCratesCollection


### Awesome Libraries and Frameworks

#### Web Development
- Rocket: [https://rocket.rs/](https://rocket.rs/): A fast and expressive web framework for Rust.
- Actix Web: [https://actix.rs/](https://actix.rs/): An asynchronous, actor-based web framework for Rust.
- Axum: [https://github.com/tokio-rs/axum](https://github.com/tokio-rs/axum): A minimalist and modern web framework for Rust.
- Zola: [https://github.com/getzola/zola](https://github.com/getzola/zola): Your one-stop static site engine.
    
#### Game Development 
- Amethyst: [https://github.com/amethyst/amethyst](https://github.com/amethyst/amethyst)
- Bevy: [https://bevyengine.org/](https://bevyengine.org/)
- Godot with Rust bindings: [https://github.com/godot-rust/gdnative](https://github.com/godot-rust/gdnative)
#### System Programming
- Tokio: [https://tokio.rs/](https://tokio.rs/) for asynchronous programming.
- Diesel: [https://github.com/diesel-rs/diesel](https://github.com/diesel-rs/diesel) for database interaction.

#### Data Science and Machine Learning
- Apache Arrow: [https://arrow.apache.org/blog/2023/02/13/rust-32.0.0/](https://arrow.apache.org/blog/2023/02/13/rust-32.0.0/)
- DataFusion: [https://arrow.apache.org/blog/2019/02/04/datafusion-donation/](https://arrow.apache.org/blog/2019/02/04/datafusion-donation/)
- TensorFlow: [https://github.com/tensorflow/rust/issues](https://github.com/tensorflow/rust/issues)
